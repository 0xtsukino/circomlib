# `Bits2Num_strict()`

## Description

This template converts a binary number `in[n]` of `n` bits to its
integer representation STRICT

## Schema

```
             _____________________     
            |                     |
in[n] ----> |  Bits2Num_strict()  | ----> out
            |_____________________|     
```
